   ```bash
   git clone https://github.com/HLPFLCG/hlpflforms.git
   cd hlpflforms
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Run Tests**
   ```bash
   # Visit test pages in browser
   open public/test.html
   open public/ux-test.html
   ```

5. **Build for Production**
   ```bash
   npm run build
   ```

### For Users

1. **Visit Application**
   - Navigate to deployed URL
   - Or run locally

   - Register with email/password
   - Or use default credentials (admin/admin123)

   - Add fields with drag-and-drop
   - Customize field properties
   - Save and publish

4. **Embed Form**
   - Copy embed code
   - Paste into your website
   - Start collecting submissions

5. **View Analytics**
   - Check dashboard for metrics
   - View individual submissions
   - Export data as needed
